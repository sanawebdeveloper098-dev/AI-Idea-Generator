# ✨ AI Idea Generator

An AI-powered idea generation web application that helps users instantly create creative and personalized ideas using Google's Gemini AI. Users can select a category, generate unique ideas, and save their favorite ideas for later.

## 📌 About The Project

AI Idea Generator is a modern web application built to explore the power of Generative AI in everyday creativity.

The application uses the Gemini AI API to generate unique ideas based on user-selected categories. It also integrates Supabase to store users' favorite ideas, creating a complete frontend + AI + database experience.

## ⚙️ How It Works

1. User selects an idea category.
2. User enters a topic or requirement.
3. JavaScript sends a request to Gemini AI using `fetch()`.
4. Gemini generates creative ideas.
5. Ideas are displayed as interactive cards.
6. Users can save favorites to Supabase.
